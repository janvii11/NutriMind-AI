# 🥗 NutriMind AI

> **AI Powered Smart Food Recognition & Nutrition Intelligence Platform**

NutriMind AI is an intelligent nutrition assistant that recognizes food from images, estimates nutritional values, calculates health scores, and helps users track their daily meals using Artificial Intelligence.

---

## 🚀 Features

* 📷 AI Food Recognition using Deep Learning
* 🤖 Gemini AI Fallback for Unknown Foods
* 🍽 Nutrition Analysis
* 🔥 Calories, Protein, Carbs & Fat Estimation
* 💚 Health Score
* 🥗 Nutri Score
* 🤖 AI Nutrition Coach
* ⚠️ Meal Risk Analysis
* 🍴 Portion Estimation
* 📊 Interactive Dashboard
* 📝 Meal History Tracker
* 👤 User Profile

---
